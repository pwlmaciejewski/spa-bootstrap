# SPA Bootstrap

Opinionated Single Page App bootstrap project.

## Tech stack

* Typescript
* React.js
* React-Router
* Redux
* Redux-Thunk
* SemanticUI
* Stylus
* CSS Modules
* Webpack
* Jest
* Enzyme

## Deployment

It uses CircleCI to deploy the project on Heroku.

CircleCI variables:

* HEROKU_API_KEY
* HEROKU_PROJECT_NAME
